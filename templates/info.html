This website was created for a need that i had. i am a creative person, and i like to try new things. I used to search in YouTube for DIY videos but after a while,it got really confusing. i found out that it is a lack of order in the ideos and there are no spicific tips that they give, so i desided to make this website, for any creative person or anyone who want to know more about uses of different tools!
The only functional categoty for now is the "School Suplies", but all categories are going to be accessible soon, and creat more categories to discover.
The MVP of this website can be given only tips(text), but it will be able to receive photos and videos soon.
